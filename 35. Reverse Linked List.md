### Description
https://www.lintcode.com/problem/reverse-linked-list/description \
Reverse a linked list.

### Example
Example 1:\
Input: 1->2->3->null\
Output: 3->2->1->null

Example 2:\
Input: 1->2->3->4->null\
Output: 4->3->2->1->null

### Code
```
class Solution:

    def reverse(self, head):
        #curt表示前继节点
        curt = None
        while head != None:
            #temp记录下一个节点，head是当前节点
            temp = head.next
            head.next = curt
            curt = head
            head = temp
        return curt
```
