Reverse a linked list.

Have you met this question in a real interview?  
Example
Example 1:

Input: 1->2->3->null
Output: 3->2->1->null
Example 2:

Input: 1->2->3->4->null
Output: 4->3->2->1->null
Challenge
Reverse it in-place and in one-pass

Related Problems

```
class Solution:

    def reverse(self, head):
        #curt表示前继节点
        curt = None
        while head != None:
            #temp记录下一个节点，head是当前节点
            temp = head.next
            head.next = curt
            curt = head
            head = temp
        return curt
```
